# Tripleten web_project_homeland

Titulos de Patria a Patria

## Descripción

Sitio web responsivo que presenta una galería de ciudades empleagos. El proyecto mostrara diferentes lugares del mundo con sus historias, imágenes y características únicas.

## Autor

Carlos Ignacio Romero Durán

## Tecnologías utilizadas

*HTML5
*CSS
*Flexbox
*Media Queries
*Metodología BEM
*Fuentes locales (Inter)

## Características del proyecto

*Diseño responsivo para desktop (1280px+), tablet (768px-1024px) y móvil (320px-768px)
*Implementación de CSS y Flexbox
*Uso de fuentes locales para mejor rendimiento
*Metodología BEM para organización del CSS
\*Optimización de imágenes para diferentes dispositivos
